Kết nối tài khoản Facebook

Cần tạo Facebook App (miễn phí) để đăng nhập. Làm 1 lần duy nhất:
1. Vào developers.facebook.com/apps → Create App
2. Chọn loại Business → Điền tên app → Create
3. Vào Settings → Basic → Copy App ID
4. Vào Facebook Login → Settings → Thêm http://localhost:3000 vào Valid OAuth Redirect URIs
Lấy tại Facebook Developers → Your App → Settings → Basic → App ID
Tool phải chạy qua http://localhost:3000 (không phải file:///) để Facebook Login hoạt động. Nhấn đúp vào file start-server.bat trong thư mục SoraTools để khởi động.

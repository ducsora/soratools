Hướng dẫn sử dụng

1
Cấu hình chung – Điền Campaign ID (nếu dùng campaign cũ), Ad Account, Pixel, Page ID và các giá trị mặc định.
2
Bảng Ad Sets/Ads – Mỗi hàng = 1 Ad Set + 1 Ad. Thêm hàng, nhân bản để tạo nhanh. Chỉnh sửa trực tiếp trên bảng.
3
Xóa ID để tạo mới – Nếu muốn tạo Ad Set mới trong campaign cũ, để trống cột "Ad Set ID". Giữ Campaign ID nếu muốn gộp vào campaign cũ.
4
Preview & Export – Xem trước bảng Excel, kiểm tra dữ liệu rồi nhấn "Xuất .xlsx". File xuất ra đúng chuẩn cột Facebook.
5
Import vào Ads Manager – Vào Ads Manager → Nhập/Xuất → Import Ads (.xlsx). Hệ thống sẽ hiển thị các mục dưới dạng bản nháp để bạn review rồi đăng.
